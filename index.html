Relative pitch training-chord-progression

How to use

First, click the 'Hear I Chord' button.

Next, click the 'Start' button to listen to the chord progression."

Finally, type the chord progression you heard using the following input method.

The input method

Degree Numbers: Represent each chord using its degree number (e.g., 1 for I, 2 for II).

Minor Chords: Add a lowercase "m" immediately after the number (e.g., 6m, 2m).

Separators: Separate each chord with a forward slash (/).

Example:For the progression C => Am => Dm => G, type it as:1/6m/2m/5
